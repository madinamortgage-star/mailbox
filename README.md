# Website UI kit

Marketing site for **My Biz Mailbox**, rendered in the Apple-inspired design language.

## Components
- `Nav.jsx` — translucent sticky nav (glass)
- `Hero.jsx` — full-bleed hero section with headline + pill CTAs
- `ProductTile.jsx` — the two-up product card in section grids
- `FeatureGrid.jsx` — 2-column or 3-column tile arrangement
- `PromoStrip.jsx` — the thin highlight band ("New" announcement)
- `Footer.jsx` — Apple-style compact link footer

## How to use
Open `index.html` to see a live click-thru of the full marketing page. Each section is a React component loading via Babel.

The page uses the Apple section-rhythm:
1. Glass nav
2. Black hero — product as sculpture
3. Light gray hero — inverse
4. 2-up feature tiles
5. 3-up secondary tiles
6. CTA strip
7. Footer
